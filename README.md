# vermilion_public

Public, reusable Claude Agent Skills from Vermilion.

## Skills

| Skill | What it does |
| --- | --- |
| [`launch-playbook`](skills/launch-playbook/) | Vermilion's tiered product-launch and funding-announcement playbook for portfolio companies — tier model, per-tier checklists with owners and timing, a funding-announcement flow, and bundled messaging/PR/Product Hunt templates. |

## Installing a skill

Each skill is a self-contained folder under `skills/`. Copy the folder into your agent's skills directory, or package it into a `.skill` file.
