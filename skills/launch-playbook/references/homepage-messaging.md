# Homepage messaging

Supports the **Site** section (any tier): updating homepage / landing-page copy when a launch changes what the site should say.

**Live source:** https://app.notion.com/p/2fe70e83e94d80a6ac43e33121bb06e9

The page hosts a framework from **Fletch** — a solid way to think through product messaging and homepage copy — as an attached PDF ("Fletch's 4-Step Process") plus a homepage template image. The attachments live on the Notion page; open the live source to view or download them.

Use this when a release or raise means the homepage hero, sub-head, or section copy needs to change (Tier 3 "Homepage updated," Funding "Add banner to site"). The Fletch process walks from positioning → value props → proof → page structure, so the homepage tells the same story as the launch one-pager rather than drifting from it.
